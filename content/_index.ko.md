# 컨테이너를 활용한 Aerial Framework 및 Sionna 통신 연구 워크샵

이 워크샵에서는 NVIDIA Sionna와 Aerial Framework를 활용한 Docker 컨테이너 및 Git 버전 관리 기반의 재현 가능한 통신 시뮬레이션 연구 환경을 구축하는 방법을 학습합니다.

## 워크샵 개요

### 대상
이 워크샵은 **통신 연구자**를 대상으로 합니다. 특히 다음과 같은 연구자들에게 적합합니다:
- 무선 통신 시스템 연구자
- 5G/6G 통신 기술 개발자
- 통신 신호 처리 및 AI/ML 연구자
- 통신 시뮬레이션 환경 구축이 필요한 연구팀

### 배경
현대 통신 시스템 연구는 복잡한 AI/ML 워크로드와 대규모 시뮬레이션을 필요로 합니다. 이 워크샵에서는 Docker 컨테이너 기술과 Git 버전 관리를 통해 NVIDIA Sionna와 Aerial Framework 기반의 재현 가능하고 협업 가능한 연구 환경을 구축하는 실습 중심의 학습을 제공합니다.

## 학습 목표

- Docker 컨테이너 기술을 활용한 재현 가능한 실험 환경 구축
- NVIDIA Sionna 및 Aerial Framework 기반 통신 시뮬레이션 환경 컨테이너화
- AI-DLC(Development Life Cycle) 개념을 바탕으로 한 Git 버전 관리 시스템 구축
- Docker Compose를 활용한 통합 개발 환경 및 협업 워크플로우 구현
- 실험 변경사항 추적 및 재현성을 보장하는 연구 환경 표준화

## 사전 요구사항

- Docker 컨테이너 기본 개념 및 경험 (선호사항)
- Git 버전 관리 기본 지식 (선호사항)
- 네트워킹 개념 이해 (권장사항)
- NVIDIA 프레임워크 기본 지식 (권장사항)
- Kubernetes 기초 (선택사항)

## 예상 소요 시간

약 4시간

## 워크샵 구성

1. **사전 준비** - 환경 설정 및 구성
2. **소개** - 통신 연구 과제 및 컨테이너 솔루션 개요
3. **Docker 기초** - Docker 컨테이너 기술 및 Sionna 환경 구축
4. **컨테이너 기반 실험 격리 환경** - TensorFlow 통합 연구 환경 구성 with Aerial Framework
5. **Git 버전 관리** - AI-DLC 기반 실험 코드 및 데이터 관리
6. **통합 실험 환경 및 협업 워크플로우** - 실험 변경사항 추적 및 재현성 구현
8. **요약** - 실습 산출물 정리 및 확장 학습 방향
