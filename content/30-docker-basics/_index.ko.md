# Docker 기초

클라우드 리소스를 효율적으로 활용하는 것은 통신 시뮬레이션 연구를 포함, IT 인프라를 활용하는 여러 환경에 있어 핵심이 되었습니다. 이 모듈에서는 Docker 컨테이너 기술을 활용하는 방법을 배워봅니다.

## 🎯 학습 목표

이 모듈을 완료하면 다음을 할 수 있습니다:

- 컨테이너와 가상머신의 차이점 및 Docker 아키텍처 이해
- Docker 기본 명령어를 사용한 컨테이너 실행 및 관리
- Dockerfile을 작성하여 커스텀 이미지 빌드
- Volume을 활용한 데이터 영속성 관리
- Multi-stage 빌드를 통한 이미지 최적화
